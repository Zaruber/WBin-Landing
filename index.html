Возможности

Всё для аналитики в одном расширении

Каждый модуль работает на ваших страницах кабинета WB и забирает данные напрямую.

1

Сбор кампаний

Извлечение данных о рекламных кампаниях и артикулах. Поддержка API v5 и v6, автоматическое зеркалирование запросов.

2

Сбор статистики

Анализ эффективности рекламы: детальная статистика по каждой кампании, показы, клики и конверсии.

3

Анализ АСК NEW

Ассоциированные конверсии в отдельной вкладке: поддержка множества ID кампаний, агрегация и копирование результатов.

4

Финотчёты + хранение NEW

Анализ финансовых отчётов и платного хранения с агрегацией по артикулам — результат копируется в буфер для вставки в Google Sheets.

Как работает

Три шага до первого отчёта

Установите расширение

Один клик в Chrome Web Store — иконка WBin появится на панели браузера.

Откройте кабинет WB

Перейдите на страницу кампаний, статистики или финотчётов в личном кабинете Wildberries.

Нажмите «Получить отчёт»

Данные собираются в готовую таблицу прямо в браузере — копируете в буфер и вставляете в Google Sheets через Ctrl+V. Без скачивания файлов.
