# WBin

Расширение для браузера для аналитики Wildberries. Помогает продавцам и менеджерам выгружать данные рекламных кампаний, статистику и финансовые отчёты из личного кабинета Wildberries и переносить их в таблицы за пару кликов — без ручного копирования и сторонних сервисов.

**Установить:** [Chrome Web Store](https://chromewebstore.google.com/detail/dkpbihfjchandhkllkmljeijnlignhpk?utm_source=item-share-cb)
**Сообщество:** [Telegram](https://t.me/+3g9fjC4OTn03MWYx)

## Возможности

- **Сбор кампаний** — выгрузка данных о рекламных кампаниях и артикулах. Поддержка API v5 и v6.
- **Сбор статистики** — детальная статистика по кампаниям: показы, клики, конверсии.
- **Анализ АСК** — ассоциированные конверсии в отдельной вкладке, агрегация по нескольким кампаниям.
- **Финотчёты и платное хранение** — анализ финансовых отчётов с агрегацией по артикулам.

Все результаты копируются в буфер обмена для вставки в Google Sheets через Ctrl+C / Ctrl+V — работа целиком в браузере, без скачивания файлов.

## Как работает

1. Установите расширение из Chrome Web Store.
2. Откройте нужную страницу в личном кабинете Wildberries (кампании, статистика, финотчёты).
3. Нажмите «Получить отчёт» — данные соберутся в готовую таблицу, которую можно скопировать в Google Sheets.

## Конфиденциальность

Расширение собирает только анонимную статистику использования для улучшения интерфейса и не передаёт персональные данные, идентификаторы аккаунтов или содержимое кабинета продавца. Подробнее — в [политике конфиденциальности](https://zaruber.github.io/WBin-Landing/privacy.html).
